2018年第二期就业指导

1. 企业需要什么样的人才？

面试不仅仅需要的是技术，还有其它很多东西，我们把它称为一个人的 “综合能力” 或者 “职业精神”

职业技能：技术

  • 有干活能力
    • 当前的项目经验
      • 遇到问题是如何解决的
      • 是否有大型项目的研发经验
      • 是否自己是主负责人
      • 对项目的突出贡献
    • 技术基本功
      • 出一些面试题（出好的让你做，以及临时出的题）
      • 问一些基础和常用的语法（目前以ES6 / REACT / VUE / 移动端开发为主）

你们的简历上编写的项目，要求必须自己熟练

  1. 技术栈（包含使用的插件）
  2. 项目的功能板块和流程，并且知道怎么去做
  3. 会遇到哪些问题，我们应该怎么解决（突出自己的贡献很大，力挽狂澜）

基础知识：多做面试题

  • 有干好活的能力

    • 原理：mvc/mvvm/redux/jsx/Promise/跨域…
    • HTTP：三次握手和四次挥手、304缓存、HTTP2、GET和POST、常用的报文、常用的状态码
    • 浏览器渲染机制：同步编程和异步编程（微任务和宏任务）、闭包、堆栈、词法解析等
    • 算法：常规算法（冒泡、快速、插入、去重、递归…）
    • 性能优化

  • 有以后能干更好的潜力

    • node / webpack（突出的是性能优化）
    • 对新技术的热爱与痴迷：canvas / webGL
      • 学习能力强
      • 上进心
    • 技术博客论坛：发帖或者自己贡献一些东西（博客）

职业素养：人品

  • 沟通能力
    • 团队融入
    • 擅长交流
  • 吃苦精神
    • 在学习上的吃苦
    • 工作上的努力
    • 抗压能力强
    • 任劳任怨
    • 上进心
  • 有想法
    • 未来职业规划
  • 最好具备一定的领导力

  1. 面试过程中，遇到不会的问题？

    • 拿出笔和纸，当着面试官记下来，可以让面试官仔细和你说
    • 如果一点都没听过，问问面试官是干啥的，如果和你知道的一个很相似，一定要和面试官说，自己会哪个
    • 如果会一些，可以在某些话语上，表现出自己的自信，例如：和XXX很相似，简单，晚上回去搞定

  2. 面试的时候带上一本书

    • 面试途中，静心看书，克服紧张心理（破罐子破摔）
    • 让面试官觉的你爱学

  3. 提前准备好自我介绍和一些常见面试题的回答

    • 沟通、表达
    • 逻辑思维的
    • 突出自己擅长的部分

  4. 非技术类的面试题

    • 离职原因
    • 职业规划
    • 自我优缺点评价
    • 如何看待加班
    • …

2. 要求

  1. 在第二周实战周结束之前，把简历都发给我审核通过
  2. 即使毕业了，建议大家来学校自习（团结）
  3. 把面试题拍下来（或者总结后的）发群里
  4. 电话不允许关机
  5. 不允许放弃：面试是一个综合体、运气、颜值、努力综合于一身的，只要不放弃，100%能找到工作，而且薪资都能达到你的期望
